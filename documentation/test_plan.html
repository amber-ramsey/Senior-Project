Test Case Pre Conditions Test Steps Post Conditions Expected Results Actual Results Pass/Fail
User View
  • User is logged in to their account
  • User has taken at least one test
  • User clicks the View Test Results button in the navigation bar
  • User is redirected to the Tests Taken Page
  • User will scroll through the tests they have taken
  • User clicks the View Results button on the test they choose
  • User is redirected to the View Results page for that test
  • User is able to see only their test score along with any information the test creator added for the test taker to see in this view
  • Redirected to View Results page for the test
  • User sees only test results
Admin View
  • Admin is logged in to their account
  • At least one test is created
  • Test has been taken by at least one person
  • Admin is located on the home page
  • Admin clicks on view results for one of the tests
  • Admin is redirected to the Test Results page for the test
  • Admin is able to see all the users who have taken the test along with their results
  • Admin can see overall tests results for everyone who has taken the test
  • Admin is redirected
  • Admin sees the users who have taken the test and their results
  • Admin sees overall tests results
